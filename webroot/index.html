Cloudflare Tunnel

Cloudflare Tunnel memungkinkan Anda menghubungkan container atau server lokal ke internet dengan aman tanpa membuka port publik. Terdapat dua jenis utama:

  • Quick Tunnel — tidak memerlukan login, cocok untuk uji coba cepat. URL bersifat sementara.
  • Named Tunnel — membutuhkan login Cloudflare dan konfigurasi domain untuk penggunaan permanen.

Cara penggunaan dasar:

  • Login ke Cloudflare: cloudflared login
  • Buat tunnel baru: cloudflared tunnel create nama_tunnel
  • Konfigurasi domain: cloudflared tunnel route dns nama_tunnel domain.com
  • Jalankan tunnel: cloudflared tunnel run nama_tunnel

Dengan konfigurasi ini, container Anda dapat diakses secara publik melalui domain Cloudflare tanpa membuka port secara langsung.

PM2 — Process Manager

PM2 digunakan untuk mengelola aplikasi Node.js agar tetap berjalan stabil di background dan otomatis restart setelah container reboot.

  • pm2 start app.js --name my-app
  • pm2 ls
  • pm2 logs my-app
  • pm2 stop my-app
  • pm2 save dan pm2 startup

PM2 menjaga proses Node.js agar selalu aktif dengan manajemen log yang efisien.

Google Chrome / Puppeteer Path

Container ini menyertakan Google Chrome Stable agar dapat digunakan oleh Puppeteer untuk automasi dan rendering halaman web. Path default Chrome dapat digunakan langsung:

/usr/bin/google-chrome-stable

Path ini digunakan dengan variabel PUPPETEER_EXECUTABLE_PATH di aplikasi Node.js Anda.
